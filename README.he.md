# Hebrew RTL Helper

[English](README.md)

תוסף Chrome קטן שמשפר קריאות של עברית וכיווניות RTL בדפי אינטרנט. הוא פועל רק באתרים שתבחרו.

התוסף מזהה טקסט עברי באזורים נפוצים של תוכן, מחיל כיוון RTL לפי ההגדרות, ומשפר טיפוגרפיה בעברית, ריווח, וקריאות של טקסטים מעורבים בעברית, אנגלית וקטעי קוד.

## יכולות

- זיהוי אוטומטי של טקסט עברי.
- החלת כיוון RTL רק על אזורי תוכן רלוונטיים.
- שמירה על מאפייני `dir` ו־`lang` קיימים.
- הימנעות משדות קלט, אזורים עריכים, ניווט, סיידברים, סקריפטים, סגנונות, SVG ובלוקי קוד.
- שיפור fallback של פונטים בעברית עם Segoe UI, Noto Sans Hebrew, Noto Sans, Arial ופונטים של המערכת.
- שיפור קריאות של Markdown, אנגלית ו־inline code בתוך טקסט עברי.
- פתיחת פופאפ הגדרות ממותג בלחיצה על אייקון התוסף.
- שמירת בחירות המשתמש באמצעות `chrome.storage`.
- החלה מחדש על הדף הנוכחי אחרי שינוי דינמי של הדף או אחרי תרגום.

## הגדרות

אפשרויות התיקון מוכנות כברירת מחדל. בוחרים באילו אתרים להפעיל את התוסף:

- הפעל באתר הזה
- כיוון RTL
- יישור Markdown וקוד
- שינוי גופן
- ריווח וקריאות

טיפ: אחרי תרגום דף עם Google Translate, לחצו שוב על **החל בדף הזה**.

## התקנה מקומית

1. פותחים את `chrome://extensions`.
2. מפעילים Developer mode.
3. לוחצים על **Load unpacked**.
4. בוחרים את תיקיית הפרויקט.

## בדיקות

התוסף גנרי ויכול לרוץ על כל דף אינטרנט. דפים דינמיים עם תוכן עברי מוזרם או מתעדכן לעיתים קרובות הם מקום טוב לבדוק בו את התנהגות ה־RTL.
